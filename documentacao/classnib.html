Detailed Description

Numero de Identificao Bancaria (21 Digits) - Format BBBB AAAA NNNN NNNN NNNC C.

The NIB is a 21 lenght number that contains the Bank, the Agency and the Bank Account Number and it has a control digit To compute the digit control we need to do the remaind by 97 of the NIB number without the last 2 digits and then subtract from 98 the result
